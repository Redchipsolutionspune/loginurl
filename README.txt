The following steps should get you up and running with
this block template code.

* DO NOT PANIC!

* Unzip the archive and read this file

* Place the loginurl folder into the /block folder of the moodle
directory.

* Modify version.php and set the initial version of you module.

* Visit Settings > Site Administration > Notifications, you should find
the module's tables successfully created

* Go to Site Administration > Plugins > Blocks > Manage blocks
and you should find that this loginurl has been added to the list of
installed modules.

Good luck!
